Neste repositório, aplico conceitos de lógica de programação, manipulação de dados e algoritmos através dos seguintes desafios:

### 🧠 Lógica e Algoritmos
- **99 Bottles of Beer**: Prática de estruturas de repetição (`while`/`for`) para gerar letras de músicas dinamicamente.
- **Fibonacci Challenge**: Implementação da sequência numérica de Fibonacci usando loops ou recursão para entender crescimento exponencial.
- **FizzBuzz Challenge**: O clássico desafio de divisibilidade para praticar operadores de módulo (`%`) e condicionais aninhadas.
- **Leap Year Checker**: Algoritmo para validar anos bissextos com base em múltiplas condições lógicas.
- **Espelhar String**: Manipulação de cadeias de caracteres para inverter textos.

### 🔢 Cálculos e Matemática
- **Calculadora de Peso (BMI)**: Função que calcula o Índice de Massa Corporal e classifica o resultado (Baixo peso, Normal, etc.).
- **Life in Days, Weeks, Months**: Cálculo de tempo de vida restante com base em uma idade alvo, praticando operações matemáticas básicas.
- **Love Match Calculator**: Gerador de "compatibilidade" aleatória ou baseada em nomes para praticar `Math.random()` e manipulação de strings.

### 💻 JavaScript Funcional e Objetos
- **High-Order Functions**: Uso de funções que recebem outras funções como argumento (ex: calculadora funcional).
- **Constructor Functions**: Criação de moldes para objetos (como o exemplo `empregadaDomestica`) para entender o sistema de protótipos do JS.
- **Callbacks**: Estudo de funções de retorno e sua importância em eventos de clique e teclado.

### 🔠 Manipulação de Texto
- **Contador de Caracteres (Twitter)**: Lógica para limitar e contar caracteres de entrada de usuário.
- **Slicing & Case Formatting**: Uso de métodos como `.slice()`, `.toUpperCase()` e `.toLowerCase()` para formatar nomes de usuários.
- **Check Name Includes**: Validação de presença de termos específicos em strings usando o método `.includes()`.
